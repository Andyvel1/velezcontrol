How it works

Three ways to use it

The VCS-SYS1 is designed to be useful the moment you power it on, but flexible enough for any integration.

01

Standalone

Power it on and it runs. The onboard MCU captures audio, decomposes it into bass, mids, and highs, and maps each band to a PWM channel. No external controller required.

02

Raspberry Pi 5 HAT

Mounts directly onto a Pi 5 via the 40-pin header. The board powers the Pi from its 3.3V rail. Install jumpers to route Pi PWM/GPIO lines to the gate drivers for host-controlled output — or leave them off and use the Pi purely as a compute platform for edge AI, wireless control, or automation.

03

External control

Feed the board PWM and GPIO signals from any microcontroller via jumper-selectable headers. The board provides 3.3V regulated power back to your controller. That's the entire interface.

04

Custom firmware

Flash your own code onto the MSPM0G3507 over USB-C. Full peripheral access. Stock firmware is open source under MIT license with a complete CCS project.

External control interface

Simple connection

2 PWM pins, 2 GPIO pins, power, and ground. Any microcontroller with digital outputs can drive all four channels.
